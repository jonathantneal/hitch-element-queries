Element Queries with Hitch.js

Element Queries are a way for you to style something based on its measurements. Where Media Queries care about the size of the screen, Element Queries care about the size of the thing.

.inner-bar { background: limegreen; }
.inner-bar:max-width(30rem) { background-color: royalblue; }
.inner-bar:min-width(60rem) { background-color: red; }
